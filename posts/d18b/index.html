Java 是一种强类型语言，每个变量都必须声明其数据类型。
一、基本数据类型（primitive data type）
数值型
整数类型：byte, short, int, long
浮点类型：float, double
字符型：char
布尔型：boolean

计算机内存的最小存储单元是字节（byte），一个字节就是一个8位二进制数，即8个bit。它的二进制表示范围从0000000011111111，换算成十进制是0255

对于整型类型，Java只定义了带符号的整型，因此，最高位的bit表示符号位（0表示正数，1表示负数）。各种整型能表示的最大范围如下：

byte：-128 ~ 127
short: -32768 ~ 32767
int: -2147483648 ~ 2147483647
long: -9223372036854775808 ~ 9223372036854775807


二、引用数据类型（reference data type）
引用类型就是：变量保存的是对象在内存中的地址（引用），而不是对象本身的值。
类（class）
接口（interface）
数组（array）

引用类型最常用的就是String字符串：
String s = “hello”;
引用类型的变量类似于C语言的指针，它内部存储一个“地址”，指向某个对象在内存的位置


常量：
定义变量的时候，如果加上final修饰符，这个变量就变成了常量：
final double PI = 3.14; // PI是一个常量


自增和自减
++ –
++n表示先加1再引用n，n++表示先引用n再加1

例子：

public class test &#123;
    public static void main(String[] args) &#123;
        int n = 3300;
        n++; // 3301, 相当于 n = n + 1;
        n--; // 3300, 相当于 n = n - 1;
        int y = 100 + (++n); // 不要这么写
        System.out.println(y);
    &#125;

switch表达式
Java 14+ 的 新式 switch 表达式写法
-> 箭头语法
防止 case 穿透 每个 case 自动结束，不会掉进下一个 case

// switch
public class Main &#123;
    public static void main(String[] args) &#123;
        String fruit = "apple";
        switch (fruit) &#123;
        case "apple" -> System.out.println("Selected apple");
        case "pear" -> System.out.println("Selected pear");
        case "mango" -> &#123;
            System.out.println("Selected mango");
            System.out.println("Good choice!");
        &#125;
        default -> System.out.println("No fruit selected");
        &#125;
    &#125;
